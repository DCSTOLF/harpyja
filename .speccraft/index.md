# Harpyja

A read-only, offline MCP server that turns a natural-language query into exact `file:line` citations across large/legacy/air-gapped codebases.

## Stack

- Python 3.12+
- FastMCP (MCP server; stdio + streamable HTTP)
- Tree-sitter (symbol layer: Go, Rust, Python, JS/TS, C#, Java, C/C++) + ripgrep fallback
- Tier 1 "Scout": native explorer loop (a general OpenAI-compatible tool-calling model over five read-only tools `{grep,glob,read_span,ls,symbols}`) â€” as of spec 0024 (+`ls` spec 0027, +`symbols` spec 0030); FastContext is FULLY REMOVED as of spec 0025 (no external finder dependency; the explorer is the sole Scout backend, running on `lm_model`)
- DSPy `dspy.RLM` over Deno/Pyodide WASM sandbox (Tier 2 "Deep")
- Local OpenAI-compatible model endpoint: llama.cpp (`llama-server`) or Ollama. As of spec 0025 the default `lm_model` (`hf.co/Qwen/Qwen3-8B-GGUF:latest`) is the wired single model for BOTH the Scout explorer loop and Deep; any OpenAI-compatible tool-calling model is swappable. âš ď¸Ź **The default tag is NOT servable on the live stack**: spec 0036 verified `hf.co/Qwen/Qwen3-8B-GGUF:latest` does not serve on the dev Ollama (it was the AC8 arm-A swapped out for the servable `qwen3:14b`) â€” a live run must set `lm_model` to a served tag via toml/env/`--deep-model`; the default is a provisional placeholder, not a validated live target. âš ď¸Ź **8 GB footprint NOT validated**: an 8B-class model co-loaded with Deep + the Deno/Pyodide sandbox under `mode=auto` exceeds a small GPU â€” treat "8 GB" as aspirational, not a validated minimum
- `uv` for env/deps

## Architecture in one paragraph

A FastMCP server exposes three tools (`locate` / `read` / `index`) and holds no business logic. The Orchestrator classifies each query, then runs the cheapest tier that can answer: Tier 0 (deterministic tree-sitter symbols + ripgrep), Tier 1 (Scout, a native tool-calling explorer loop), and Tier 2 (Deep, a `dspy.RLM` over bounded read-only host tools). A Verification Gate reads cited lines back and escalates on failure; a Citation Formatter merges, ranks, and clamps results. The Model Gateway is the single outbound abstraction, pointed only at localhost â€” the one place the air-gap guarantee is enforced. Every layer has a fallback so retrieval never goes blind.

## Hard rules (see guardrails.md)

- No network egress at runtime â€” model/search/parsing stay local; only the Model Gateway calls out, and only to localhost.
- Read-only â€” Harpyja returns citations and never edits, writes, or generates code in the target repo.
- Always degrade gracefully â€” no parser â†’ ripgrep; no model â†’ Tier 0; escalation fails â†’ best-effort Tier 1 with a confidence flag.
